How does Light affect plant's growth?

Controlled variable
  1. light duration - The amount of line out on should be the same amount
  2. type of soil
  3. amount of water
  4. temperature
  5. pot size
  6. number of seeds
  7. Light intestity - The strongness and the brightness should be kept the same
  8. Light quality - Blue light is suppsed to have a stronger steam but shorter than Red
  9. Light levels - Light levels should be the same

Steps to do
  1. Get the exact same amount of weight, light, and materials for each plant you are growing.
  2. Next, plant them in the same type and size of pots using the same amount of soil
  3. Place all plants at the same distance from their light source
  4. Keep all plants in the same environment so the temperature stays the same
  5. Give each plant the same amount of light time each day (same number of hours)
  6. Block any other light so only the coloured light affects the plants
  7. Finally, measure their growth in the same way each time

pot box colour changing light
